Accessibility Form Agent

Bridging the gap between paper and digital. Complete complex web forms from document photos with AI-guided assistance and human-in-the-loop control.

Our Mission

  • Reduce friction for screen reader and motor aid users.
  • Lower cognitive load for ADHD and executive function challenges.
  • Ensure user control, privacy, and full auditability.

Key Features

  • Parse paper documents and screenshots into structured data.
  • Multi-step field mapping and conditional logic handling.
  • Auto-detection of errors with intelligent self-recovery.
  • Draft approval system: you always have the final say.

Built on RexusCore

  • OpenAI Vision & Reasoning integration.
  • Robust Python agent services with page memory.
  • Seamless transport via MAUI client.

Accessibility Commitment

We target WCAG 2.2 AA standards across all our interfaces. From keyboard-first navigation to aria-live regions for status updates, our tools are built to be used by everyone.

The Outcome

Faster, more accurate form completion with absolute user control and a clear audit trail. What started as a tool for job forms has evolved into a versatile agent capable of handling any complex documentation workflow.
