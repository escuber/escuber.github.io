# RexusAi • KeyCatcher Landing (GitHub Pages)
This repo hosts a static landing page with a sticky left TOC and boxed sections.

Publish
1. Create a new public repo on GitHub (e.g., rexusai-keycatcher).
2. Upload all files from this ZIP.
3. Repo Settings → Pages → Source: Deploy from a branch.
4. Branch: main, Folder: / (root) → Save.

Customize
- Colors: edit assets/styles.css variables at the top.
- Logo: replace assets/logo-rexusai.svg and assets/logo-rexusai-icon.svg.
- CTA email: edit the mailto in index.html.
- Custom domain: add your domain in Settings → Pages and put it in a CNAME file at the repo root.
